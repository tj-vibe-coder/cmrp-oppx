Original Error:

app.js:3269 Uncaught SyntaxError: Invalid regular expression: /[\s\._-@]+/: Range out of order in character class

Root Cause: In JavaScript regular expressions, when using character classes [...], the hyphen - has special meaning as a range operator. When placed between characters like [._-], it tries to create a range from . (ASCII 46) to _ (ASCII 95), but with _ (ASCII 95) after - (ASCII 45), it creates an invalid range.
